# CollegeJS

This repository contains course materials and practice exercises for a JavaScript course.

## Repository Structure

- `Lecture1/` - Includes basic JavaScript demonstrations and examples.
- `Lecture2/` - Includes additional lecture examples.
- `MSE-1/` - Contains practice exercises and assignments.
- `temp.js` - A temporary script file.

## Course Syllabus (JavaScript Fundamentals)

### 1. Introduction to JavaScript
- What is JavaScript?
- How JavaScript runs in browsers and in Node.js
- Setting up a development environment

### 2. JavaScript Basics
- Variables and data types (`var`, `let`, `const`)
- Operators and expressions
- Statements, blocks, and comments
- Basic input/output (console.log)

### 3. Control Flow
- Conditional statements (`if`, `else if`, `else`, `switch`)
- Loops (`for`, `while`, `do...while`)
- Loop control statements (`break`, `continue`)

### 4. Functions
- Defining and calling functions
- Function parameters and return values
- Arrow functions
- Scope and closures

### 5. Objects and Arrays
- Creating and using objects
- Accessing properties and methods
- Working with arrays
- Array methods (`push`, `pop`, `shift`, `unshift`, `map`, `filter`, `reduce`)

### 6. DOM Manipulation (Basics)
- Selecting elements (`getElementById`, `querySelector`)
- Changing content and styles
- Handling events (`click`, `input`, etc.)

### 7. Error Handling and Debugging
- Common JavaScript errors
- Using `try...catch`
- Debugging with browser developer tools

### 8. Practice and Assignments
- Writing small programs to reinforce concepts
- Building simple interactive pages
- Exploring built-in APIs and libraries

## How to Use This Repository

1. Open the project in a code editor (e.g., VS Code).
2. Review the lecture files in each folder to follow along with concepts.
3. Run practice files using Node.js (if applicable) or open the HTML files in a browser.
